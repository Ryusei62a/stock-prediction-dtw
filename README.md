# DTWを用いた株価予測

このプロジェクトでは、DTW（Dynamic Time Warping）を用いて株価の類似パターンを検索し、将来の値動きを予測します。

---

## 概要

- 直近30日間の株価データを取得
- 過去の30日間のデータとDTWで類似度を計算
- 類似度が高い上位k件を取得（Top5）
- それらの「その後10日」の動きを参考に予測
- 多数決により上昇・下降を判定

---

## 手法

1. 株価データ（終値）を取得（yfinance）
2. 正規化（平均0、分散1）
3. DTWで時系列の類似度を計算
4. 類似度の高い上位k件を抽出
5. それぞれの将来の変化率を計算
6. 多数決で予測（上昇 or 下降）
